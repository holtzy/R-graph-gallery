The Retirement of rgdal

The rgdal package in R was once a cornerstone for handling spatial data. However, it has been retired (end of 2023) and is no longer maintained.
This post explains why rgdal was discontinued and discusses alternatives for spatial data analysis in R.

Quick start

The rgdal package in R was a widely used tool for reading and writing spatial data formats. However, it has been retired and is no longer maintained.

rgdal provided bindings to the Geospatial Data Abstraction Library (GDAL) for R.

Alternatives to rgdal

With the retirement of rgdal, users are encouraged to transition to more modern packages, especially sf.

sf is a powerful package for working with vector data in R. It provides a simple and consistent interface for reading, writing, and manipulating spatial data.
